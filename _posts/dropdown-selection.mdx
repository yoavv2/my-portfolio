It is well known that different browsers have a different default style for HTML elements.
When we create an application, we want to be consistent in how the app appears.

So I want to focus on the select element
and show how we can build a custom **select** element in Reactjs that will look the same in all browsers.

### The Structure

```jsx:DropDown-Structure
<DropDownContainer>
  <DropDownHeader/>
    <DropDownList>
       <ListItem/>
    </DropDownList>
</DropDownContainer>
```

```jsx:Styled-Components
const DropDownContainer = styled.div`
  width: 20em;
  margin: 0 auto;
  padding: 3em;
  display: flex;
  flex-direction: column;
  justify-content: center;
  align-items: center;
`;

const DropDownHeader = styled.div`

  padding: 0.5em;
  margin-bottom:1em
  color: black;
  font-weight: 500;
  font-size: 1.3rem;
  width: 10.5em;
  cursor: pointer;
  display: flex;
  justify-content: center;
  align-items: center;
  border: groove 6px #3faffa;
  border-radius: 1em;
`;


const DropDownList = styled.ul`
  padding: 0;
  margin: 0;
  border: groove 6px #3faffa;
  border-radius: 6%;
  box-sizing: border-box;
  color: #fff;
  font-size: 1.3rem;
  font-weight: 500;
  background-color: #fafafa;
  &:first-child {
    padding-top: 0.8em;
  }
`;
const ListItem = styled.li`
  color: black;
  list-style: none;
  width: 10em;
  border-radius: 1em;
  margin-bottom: 0.8em;
  padding: 0.5em;
  margin: 0.5em;
  cursor: pointer;
  display: flex;
  justify-content: center;
  align-items: center;
  &:hover {
    background: #3faffa;
  }
`;

```

### The States

We need 2 States :

1. a boolean state for the Opening/Closing Menu.
2. a State that will hold an object with [name] and [Color]

```jsx:states
  const [isOpen, setIsOpen] = useState(false);
  const [selectedOption, setSelectedOption] = useState({
    name: '',
    color: 'blue',
  });

```

Now we need to add an onClick event on the DropDownHeader that will toggol the isOpen state

and we want to show on the header the selectedOption:

```jsx:toggle-function
  const toggling = () => setIsOpen(!isOpen);

   <DropDownHeader
       onClick={toggling}
  >
    {selectedOption.name || 'Choose'}
   </DropDownHeader>


```

now we want to show all the option that we have, so inside the DropDownList we will map all the ListItems we have :

```jsx
<DropDownList>
  {array.map((option) => (
    <ListItem style={{ background: option.color }} key={option.name}>
      {option.name}
    </ListItem>
  ))}
</DropDownList>
```

and for the select option we will add an event on the listitem

```jsx
const onOptionClicked = (value) => {
  setSelectedOption(value);
  setIsOpen(false);
};
<ListItem
  style={{ background: option.color }}
  key={option.name}
  onClick={() => onOptionClicked(option)}
>
  {option.name}
</ListItem>;
```

and thats it.

Full code:

```jsx:DropDown.jsx
function DropDown({
  array = [
    { name: 'Apple', color: 'red' },
    { name: 'Orange', color: 'orange' },
    { name: 'banana', color: 'yellow' },
    { name: 'Mango', color: 'green' },
  ],
}) {
  const [isOpen, setIsOpen] = useState(false);
  const toggling = () => setIsOpen(!isOpen);
  const [selectedOption, setSelectedOption] = useState({
    name: '',
    color: 'blue',
  });

  const onOptionClicked = (value) => {
    setSelectedOption(value);
    setIsOpen(false);
  };
  return (
    <DropDownContainer>
      <DropDownHeader
        onClick={toggling}
        style={{
          background: selectedOption.color ? selectedOption.color : '#fff',
        }}
      >
        {selectedOption.name || 'Choose'}
      </DropDownHeader>

      {isOpen && (
        <DropDownList>
          {array.map((option) => (
            <ListItem
              style={{ background: option.color }}
              key={option.name}
              onClick={() => onOptionClicked(option)}
            >
              {option.name}
            </ListItem>
          ))}
        </DropDownList>
      )}
    </DropDownContainer>
  );
}
```
