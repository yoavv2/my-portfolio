It is well known that different browsers have a different default style for HTML elements.
When we create an application, we want to be consistent in how the app appears.

So I want to focus on the select element
and show how we can build a custom **select** element in Reactjs that will look the same in all browsers.

### The Structure

```jsx:DropDown-Structure
<DropDownContainer>
  <DropDownHeader/>
    <DropDownList>
       <ListItem/>
    </DropDownList>
</DropDownContainer>
```

```jsx:Styled-Components
const DropDownContainer = styled.div`
  width: 20em;
  margin: 0 auto;
  padding: 3em;
  display: flex;
  flex-direction: column;
  justify-content: center;
  align-items: center;
`;

const DropDownHeader = styled.div`

  padding: 0.5em;
  margin-bottom:1em
  color: black;
  font-weight: 500;
  font-size: 1.3rem;
  width: 10.5em;
  cursor: pointer;
  display: flex;
  justify-content: center;
  align-items: center;
  border: groove 6px #3faffa;
  border-radius: 1em;
`;


const DropDownList = styled.ul`
  padding: 0;
  margin: 0;
  border: groove 6px #3faffa;
  border-radius: 6%;
  box-sizing: border-box;
  color: #fff;
  font-size: 1.3rem;
  font-weight: 500;
  background-color: #fafafa;
  &:first-child {
    padding-top: 0.8em;
  }
`;
const ListItem = styled.li`
  color: black;
  list-style: none;
  width: 10em;
  border-radius: 1em;
  margin-bottom: 0.8em;
  padding: 0.5em;
  margin: 0.5em;
  cursor: pointer;
  display: flex;
  justify-content: center;
  align-items: center;
  &:hover {
    background: #3faffa;
  }
`;

```

</div>
